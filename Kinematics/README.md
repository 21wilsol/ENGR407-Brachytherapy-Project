# KINEMATICS
Both files are written in MATLAB script and should require MATLAB 2022 or later.

Forward kinematics descrubes the position of the end effector on the end of the Yaskawa robot arm.
Inverse kinematics determines the angles of the Yaskawa robot arm at any instance.

The results from these files can be used to program a robotic arm simulation and to ensure all robotic arm angles
are defined within the appropriate space.
